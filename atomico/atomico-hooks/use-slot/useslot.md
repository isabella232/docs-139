# useSlot

### Module

```javascript
import { useSlot } from "@atomico/hooks/use-slot";
```

### Syntax

```javascript
const childNodes = useSlot(ref);
```

Where:

1. `ref`: Reference of the slot to observe.
2. `childNodes`: List of nodes assigned to the observed slot.
